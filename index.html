INTRODUCTION

ECAM.css is a CSS framework that replicates the look of ECAM display elements found in modern Airbus planes.

USAGE

ECAM.css will be available via NPM, Yarn and a CDN.

-For more info.........Refer to README.md

IMPORTANT INFO

ECAM.css has no width, height or layout properties. You need to manually adjust them.

There is no font packaged with this project but usage of Source Code Pro is recommended.

Please note that the default theme is dark and no light theme is planned since the original ECAM doesn't have it.
